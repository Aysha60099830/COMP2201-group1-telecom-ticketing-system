# COMP2201 – Telecom Ticketing System  
Group 1

## Project Description

This project is part of the COMP2201 Technical Project course. Our goal is to design and implement a web-based Telecom Ticketing System that allows users to submit service requests and enables staff members to manage and update tickets in an organized way.

The system focuses on basic ticket handling features such as user login, ticket creation, ticket assignment, ticket tracking and status updates. At this stage, we are working on planning, documentation, and setting up the project structure before starting full implementation.
